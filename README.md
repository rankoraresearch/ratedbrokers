# RatedBrokers

Независимый агрегатор форекс- и криптоброкеров с прозрачной системой рейтингов.

## Быстрый старт

```bash
npm install
npm run dev        # http://localhost:5173
npm run build      # production build → dist/
```

## Стек

React 19 · Vite · react-router-dom v7 · lucide-react · fuse.js

## Масштаб

38 брокеров · 43 страны · 207 рейтингов · 25 гайдов · 4 платформы · 365+ страниц
