### Ключевые отличия от broker YAML:

| Поле брокера | Поле проп-фирмы | Комментарий |
|---|---|---|
| `regulations[]` | `legal_entity`, `regulated`, `regulatory_notes` | Проп-фирмы не имеют Tier-1/2/3 лицензий |
| `type: ECN/STP` | `challenges[]` с типами | Вместо типа аккаунта — типы challenge |
| `min_deposit` | `challenges[].accounts[].price` | Challenge fee вместо депозита |
| `spread`, `commission` | `trading_rules{}` | Правила торговли вместо спредов |
| `leverage` | `max_allocation` | Нет leverage — есть allocation |
| `accounts[]` | `challenges[].funded{}` | Funded условия вместо типов аккаунтов |
| — | `payouts{}` | Специфика выплат (у брокеров это вывод своих денег) |
| — | `trading_rules.consistency_rule` | Уникально для проп-фирм |
| — | `trading_rules.news_trading` | Критически важно для трейдеров |
| — | `challenges[].fee_refundable` | Возврат fee — важный фактор |

### Scoring System — адаптация:

**Текущий (брокеры):**
1. Regulation & Safety (25%)
2. Trading Costs (20%)
3. Trustpilot Score (15%)
4. Expert Evaluation (20%)
5. Platform & Tools (10%)
6. Execution Quality (10%)

**Предлагаемый (проп-фирмы):**
1. **Legitimacy & Trust** (25%) — возраст, юрисдикция, total payouts, прозрачность, нет ли жалоб на невыплаты
2. **Challenge Fairness** (20%) — адекватность правил, drawdown type (fixed лучше trailing), time limits, consistency rules
3. **Payout Reliability** (20%) — скорость выплат, процент успешных пейаутов, пруфы, минимальный порог
4. **Trading Conditions** (15%) — спреды, slippage, платформы, разрешённые стратегии
5. **Trustpilot Score** (10%) — аналогично брокерам
6. **Value for Money** (10%) — цена challenge vs размер аккаунта, refundable fee, free retry, scaling plan

## 5. Affiliate Economics — Prop Firms vs Brokers

| Метрика | Brokers (CPA) | Prop Firms (CPA) |
|---------|--------------|-----------------|
| Commission per signup | $200–$800 | $10–$150 (% от challenge fee) |
| Recurring revenue | Lifetime (spreads) | Per challenge purchase + resets |
| Average order value | $200+ deposit | $50–$500 challenge fee |
| Repeat purchase rate | Low | HIGH (resets, new challenges) |
| Conversion rate | 1–3% | 5–10% (lower barrier) |
| Effective CPA | $200–$800 | $50–$100 per challenge |
| Lifetime value | $500–$2000 | $200–$800 (repeat buyers) |

**Вывод:** Prop firm affiliate monetization ниже per-click, но ВЫШЕ по объёму конверсий. Трейдер покупает 3-5 challenges прежде чем пройти, плюс покупает аккаунты у нескольких фирм. Revenue per visitor может быть сопоставим с брокерами.

**Лучшие affiliate программы:**
- SabioTrade: до 70% rev share (рекорд)
- The5ers: 20-40% + lifetime
- FundedNext: 18% tiered
- ThinkCapital: до 25%
- FTMO: 15%
