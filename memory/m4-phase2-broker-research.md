## 5. Новые YAML-поля для Stock/Options/Futures

### Текущая YAML-схема (forex-centric):
```yaml
type: ECN / Market Maker      # forex execution type
min_deposit: 0
spread: "0.1"                  # forex spread
avg_spread: "0.10"
commission: $2/lot             # forex commission per lot
leverage: "1:200"
instruments: "150+"
platforms: [...]
spreads: [EUR/USD, GBP/USD...] # forex pairs comparison
```

### Новые поля — STOCKS:
```yaml
# Вертикаль stocks
stocks:
  commission_per_trade: "$0"           # US standard: $0
  commission_per_share: "$0"           # alternative pricing
  fractional_shares: true              # дробные акции
  exchanges: ["NYSE", "NASDAQ", "LSE", "Xetra"]  # доступные биржи
  exchanges_count: 50                  # кол-во бирж
  stocks_count: "10,000+"             # кол-во акций
  etfs_count: "2,000+"               # кол-во ETF
  real_stocks: true                    # настоящие акции (не CFD)
  isa_available: false                 # UK ISA (Stocks & Shares ISA)
  drip: true                          # dividend reinvestment
  extended_hours: true                 # pre/after market
  ipo_access: true                    # участие в IPO
```

### Новые поля — OPTIONS:
```yaml
# Вертикаль options
options:
  contract_fee: "$0.65"               # per contract
  exercise_fee: "$0"                  # за исполнение
  assignment_fee: "$0"                # за назначение
  strategies: ["covered calls", "spreads", "iron condors", "straddles"]
  multi_leg: true                     # multi-leg execution
  index_options: true                 # опционы на индексы
  options_screener: true              # скринер опционов
  real_time_greeks: true              # greeks в реальном времени
  paper_trading: true                 # бумажная торговля опционами
  options_level: 4                    # макс. уровень допуска (1-5)
```

### Новые поля — FUTURES:
```yaml
# Вертикаль futures
futures:
  commission_per_side: "$1.50"        # per side per contract
  micro_commission: "$0.50"           # micro contracts
  day_trade_margin: "$500"            # intraday margin E-mini
  micro_margin: "$50"                 # intraday margin micro
  exchanges: ["CME", "CBOT", "NYMEX", "COMEX", "EUREX"]
  products: ["E-mini S&P", "Micro E-mini", "Crude Oil", "Gold", "Euro FX"]
  micro_futures: true                 # micro contracts доступны
  platform_fee: "$0"                  # если платная платформа
  dom_trading: true                   # depth of market / ladder
```

### Общие новые поля (кросс-вертикаль):
```yaml
# Дополнения к общей схеме
account_types: ["Individual", "Joint", "IRA", "Roth IRA", "401k", "Trust"]
robo_advisor: true                    # робо-советник
banking_integration: true             # связка с банком
aum: "$8.5T"                         # assets under management
total_clients: "35M+"                # кол-во клиентов
founded: 1971                        # год основания (отдельно от year)
publicly_traded: "NYSE: SCHW"        # тикер на бирже
interest_on_cash: "4.5%"             # процент на неинвестированный кэш
```
