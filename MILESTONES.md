## M3. Шаблон тематического рейтинга — идеальная карточка брокера 🔄

**Статус: В РАБОТЕ**

Ключевая задача: каждая карточка брокера на странице тематического рейтинга должна содержать **уникальный SEO-текст, релевантный теме рейтинга**. Не generic "IC Markets is great", а "IC Markets is the best ECN broker because..." на странице ECN-рейтинга, и "IC Markets offers the tightest scalping spreads..." на странице scalping-рейтинга.

**Почему это критично:** Google оценивает тематическую релевантность на уровне параграфа. Если карточка брокера на странице "Best ECN Brokers" содержит тот же текст, что на "Best Brokers for Beginners" — это дублированный контент, каннибализация, и потеря позиций.

- [ ] Финализировать адаптивный шаблон BrokerRankCard (desktop / tablet / mobile)
- [ ] Определить структуру тематического текста в карточке: verdict, analysis, prosDetail, consDetail — всё привязано к теме рейтинга
- [ ] Написать тематический контент для 5 приоритетных рейтингов (×38 брокеров = 190 уникальных текстов):
  - best-forex-brokers (overall)
  - best-forex-brokers-for-beginners
  - best-ecn-forex-brokers
  - best-forex-brokers-for-scalping
  - lowest-spread-forex-brokers
- [ ] Протестировать шаблон на реальных данных: визуал, адаптивность, скорость рендера
- [ ] Утвердить финальный шаблон для масштабирования на все 207 рейтингов

## M5. Технический SEO — предпусковая оптимизация

**Статус: ПЛАНИРУЕТСЯ**

Без этого Google не проиндексирует 365+ страниц корректно.

- [ ] **sitemap.xml** — динамическая генерация для всех 365+ URL (критично для индексации)
- [ ] **robots.txt** — разрешить всё, кроме /compare/:pair (бесконечные комбинации)
- [ ] **Canonical URLs** — `<link rel="canonical">` на каждой странице (против дублей с языковыми префиксами)
- [ ] **hreflang** — подготовить для мультиязычности (когда активируем другие языки)
- [ ] **OG + Twitter meta tags** — для шеринга в соцсетях (title, description, image)
- [ ] **404 страница** — кастомная, с поиском и ссылками на популярные разделы
- [ ] **Trailing slashes** — единообразие URL (без trailing slash)
- [ ] **Скорость загрузки шрифтов** — `font-display: swap`, preload для критичных шрифтов
- [ ] **Lazy loading** — изображения, тяжёлые секции ниже fold

- [ ] **FID < 100ms** — first input delay (минимизировать JS на main thread)
